## Customize Your Badges

### Change Version

Replace `3.0.0` with your version:
```markdown
![Version](https://img.shields.io/badge/version-X.Y.Z-blue.svg)
```

### Change Color

Replace color name (e.g., `blue`, `green`, `red`):
```markdown
![Badge](https://img.shields.io/badge/text-blue.svg)
```

### Add Logo

Add `?logo=name`:
```markdown
![Rust](https://img.shields.io/badge/Rust-orange?logo=rust)
```

### Make Clickable

Wrap in `[![...](...)](link)`:
```markdown
[![Rust](https://img.shields.io/badge/Rust-orange?logo=rust)](https://www.rust-lang.org/)
```
