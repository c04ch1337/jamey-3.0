# Architectural Decision Records (ADRs)

This directory contains the architectural decision records for Jamey 3.0, documenting the key decisions made during the architectural improvement process from B+ to A+ grade.

## ADR Format

Each ADR follows the standard format:

- **Status**: Proposed, Accepted, Deprecated, or Superseded
- **Context**: The context and forces that led to the decision
- **Decision**: The decision that was made
- **Consequences**: The results and consequences of applying the decision
