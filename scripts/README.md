# Scripts

Utility scripts for Jamey 3.0 development and operations.

## Available Scripts

- **setup.sh** - Initial project setup (builds backend)
- **run.sh** - Start the backend server (kills existing process on port 3000)
- **chat.sh** - Start the CLI chat interface
- **test-conscience.sh** - Test the conscience engine via API
- **fix-mongodb-gpg-key.sh** - Fix MongoDB GPG key warning (Ubuntu)
- **check-cursor-setup.sh** - Verify Cursor IDE setup

## Usage

All scripts are executable. Run from the project root:

```bash
./scripts/setup.sh
./scripts/run.sh
./scripts/chat.sh
```
